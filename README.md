This is a web-based backgammon game against an AI opponent.
To run it you need an OpenAI API key and a local server (open with Live Server in VS Code).
Tech stack: Vanilla JS, HTML, CSS, OpenAI API (GPT-4.1)
Key features: Drag & drop, move validation, legal move calculation, AI-powered opponent
This program was created using AI prompt engineering — from basic layout to more complex problem solving and polishing with the AI agent integration. For more information check the backgammon-prompts.txt file.
